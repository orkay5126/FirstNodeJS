# node-mocha-selenium
Explore how to use Selenium with NodeJS and Mocha

## Requirements

* nodejs
  * nvm install 8.1.2
* chromedriver (http://chromedriver.storage.googleapis.com/index.html)
  * version 2.32
* node selenium webdriver
  * npm init
  * npm install --save-devselenium-webdriver
